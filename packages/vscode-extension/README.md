# Ultra Carousel Snippets

VS Code snippets for react-native-ultra-carousel.

## Available Snippets

| Prefix | Description |
|--------|-------------|
| `uc-basic` | Basic carousel setup |
| `uc-hook` | Carousel with useCarousel hook |
| `uc-autoplay` | Carousel with auto play |
| `uc-custom` | Custom animation function |
| `uc-plugin` | Create a plugin |
| `uc-import` | Import statements |
| `uc-advanced` | Advanced preset carousel |
| `uc-creative` | Creative preset carousel |

## Installation

Search "Ultra Carousel Snippets" in VS Code Extensions.
